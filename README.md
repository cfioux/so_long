## Instructions

*clone or download the minilibx*

```
git clone https://github.com/42Paris/minilibx-linux minilibx-linux
```

*make or make bonus*

```
make
```
```
make bonus
```
* usage :
```
./so_long maps/<map_name>.ber
```
or
```
./so_long_bonus maps/<map_name>.ber
```
